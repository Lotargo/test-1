# Project Agent Rules

## Read first

1. Read `docs/PLAN.md`.
2. Read `docs/DECOMPOSITION.md`.
3. Read `docs/TODO.md`.
4. Follow the task order and constraints written there.

## Working rules

1. Prefer the next unchecked task from `docs/TODO.md`.
2. Keep changes aligned with the project plan.
3. If a task is unclear, update documentation before making broad code changes.
4. Keep commits focused and explain important decisions in code or docs.
